### How to run
1. Clone repo

   ```sh
   git clone https://github.com/matheusm/pomodoro.git
   ```

2. Run it to install dependencies
   ```sh
   npm i
   ```
   
3. Run it to start the application
   ```sh
   npm run dev
   ```
    Take a look at package.json to see the availables scripts
